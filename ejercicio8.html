Crea una página web que contenga un formulario con los siguientes campos de información: Nombre, con un input de tipo texto obligatorio y con autofoco. Correo electrónico, con un input de tipo email obligatorio. URL, con un input de tipo url que muestre la ayuda "Escribe la URL de tu página web personal". Fecha, con un input de tipo date. Tiempo, con un input de tipo time. Fecha y hora, con un input de tipo datetime. Mes, con un input de tipo month. Semana, con un input de tipo week. Número, con un input de tipo number que limite la entrada a un valor entre -10 y 10. Teléfono, con un input de tipo tel. Término de búsqueda, con un input de tipo search. Color favorito, con un input de tipo color. Un botón de envío. Además, tienes que tener en cuenta los siguientes requisitos: El título de la página debe ser Formulario de prueba de HTML5.
